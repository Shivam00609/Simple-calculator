A simple command-line calculator script in Python to perform basic arithmetic operations: addition, subtraction, multiplication, division, and modulus.

## Features
- Takes two integer inputs from the user
- Performs operations based on user choice: +, -, *, /, %
- Validates operation input and returns a message for invalid choices

## Usage
1. Run the script using Python 3:
   
   python Simple-calculator.py
   
2. Input the first number when prompted.
3. Input the second number when prompted.
4. Input your choice of operation (+, -, *, /, %).
5. The script outputs the result of the operation or an error message for invalid inputs.

## Example

Enter first no.: 10
Enter second no.: 3
Enter your choice +,-,*,/,%: %
1
